# Wedding Website v6 — Gia Khoa & Thu Huyền
📅 04/04/2026 | Hải Phòng & Nam Định

## Tính năng
- Envelope animation với wax seal
- Dual-side: Nhà Trai (Hải Phòng) & Nhà Gái (Nam Định)
- Hero ảnh thật riêng cho từng side
- Countdown timer, gallery, RSVP form
- Hộp Chúc Phúc: groom side chỉ QR chú rể, bride side chỉ QR cô dâu
- Lễ Vu Quy (nhà gái) / Lễ Thành Hôn (nhà trai)
- Nearby places riêng cho từng địa điểm
- Bản đồ Google Maps

## Cấu trúc
- index.html — HTML chính
- css/main.css — Stylesheet
- js/main.js — Logic & dual-side switching
- images/hero/ — hero-groom.jpg, hero-bride.jpg
- images/gallery/ — photo-01..06.jpg

## Build file demo (all-in-one)
python3 build.py → wedding-v6-demo.html
